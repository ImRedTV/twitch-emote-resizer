Twitch Emote Resizer

View on GitHub

Drag an image anywhere on this page and it will resize the image into the sizes Twitch requires for emotes and badges. If the resized image is larger than Twitch's size limit of 25KB, this page will also attempt to compress it. You can right-click the resized images to download them as a PNG. All image processing is done client-side, no image data is ever sent anywhere else.
